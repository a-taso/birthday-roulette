# 🎂 バースデールーレット・アプリ 解説

お友達の誕生日プレゼントを楽しく決めるためのWebアプリです。
HTML, CSS, JavaScriptのみで作られており、特別なツールなしでブラウザですぐに動かせます。

## 📁 ファイル構成

- **index.html**: アプリの骨組み（入力欄、ボタン、ルーレットの表示など）
- **style.css**: デザイン（パステルポップな配色、紙吹雪アニメーション、立体的でリッチなUIなど）
- **script.js**: 動き（ルーレット制御、当選時のクラッカー演出など）

## 🚀 使い方

1. `index.html` をブラウザ（Chrome, Safariなど）で開きます。
2. 4つの入力欄に、プレゼントの候補を入力します。
3. 「スタート！」ボタンを押すと、ドキドキするアニメーションと共にルーレットが回転します。
4. 結果が決まると、紙吹雪が舞い散り「おめでとう！」と祝福されます。

## 📝 コードの役割（初心者向け）

### HTML (`index.html`)

建物の「骨組み」のようなものです。

- `<input>` タグでプレゼントの名前を入力する場所を作っています。
- `<div class="wheel">` がルーレットの本体です。

### CSS (`style.css`)

建物の「内装・外装」のようなものです。

- `background: linear-gradient(...)` で背景を綺麗なグラデーションにしています。
- `.wheel` の `border-radius: 50%` で四角い箱を丸くしてルーレットにしています。
- `transform: rotate(...)` という命令で、ルーレットを回転させます。

### JavaScript (`script.js`)

建物の中の「電気・設備」のようなものです。

- ボタンが押されたことを検知します (`addEventListener`)。
- ランダムな数字を作って、その分だけルーレットを回します (`Math.random()`)。
- どのプレゼントが選ばれたかを計算して、画面に「おめでとう！」と表示します。
